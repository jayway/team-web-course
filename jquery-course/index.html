jQuery Course

In this directory you will find a number of Javascript files along with some tests. The object of this lab is to get all the tests to pass.

Installation

The Ajax Tests give better error messages when run through a server. If you don't have one, we have provided one. To get it to work

  • Install Node.js
  • npm install
  • ./serve
  • http://localhost:4000

Lab Instructions

  • Open up the Javascript files in an editor.
  • Edit the source files, *not the tests*, until all the tests pass.
  • Detailed instructions are in the source files.
